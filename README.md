# 無國界醫生
實體機台活動和網路分享

瀏覽器同步網址
[大螢幕(長)](https://msf.lesca.net/lg.html)
[小螢幕(寬)](https://msf.lesca.net/sm.html)

### pre-install
```
$ npm i webpack -g 

$ npm i webpack-cli -g

$ npm i webpack-dev-server -g

$ npm i node-pre-gyp -g
```

### 18隻影片存放位置

[影片下載](http://msf.lesca.net/_vid/) 然後放下面資料夾裡
```
/src/lg/vid/
```

### test
```
$ npm run dev
```

#### 大螢幕本機測試 http://localhost:8080/lg.html
#### 小螢幕本機測試 http://localhost:8080/sm.html
